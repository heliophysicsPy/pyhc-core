# pyhc-core

`pyhc-core` is a package that simplifies the installation of the core PyHC packages.

## Installation

You can install the latest version of pyhc-core using pip:

```sh
pip install pyhc-core
```

## Core Packages

The following packages are included:
- hapiclient
- kamodo
- plasmapy
- pysat
- pyspedas
- spacepy
- sunpy
